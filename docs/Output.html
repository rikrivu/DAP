Class: Output

Output()

Describes the Ouput Window on the nimbus EDA webpage. Describes the layout of the Output Window - where all the components are present in the layout. Tab component is used to visualize the Excel file data as tables. A download button is present on the bottom which gives user a choice to download the Excel file.
